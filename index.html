01

Install Petra Wallet

Download the Petra Wallet Chrome extension. Create a new wallet and save your seed phrase somewhere secure — never share it with anyone.

↗ petra.app
02

Claim Testnet Tokens

You need Aptos testnet tokens to interact on-chain. Use the faucet + CLI tools from the official Shelby docs to get test tokens sent to your Petra wallet.

↗ docs.shelby.xyz/tools/cli
03

Open Shelby Explorer

Navigate to the Shelby Explorer and connect your Petra wallet to enter the devnet environment.

↗ explorer.shelby.xyz/shelbynet
04

Link Your Account

Click "View Account" in the top-right corner of the explorer. This registers your wallet on Shelby Devnet and ties your address to on-chain activity.

05

Upload a File Most Important

Upload files to generate on-chain storage interactions — the core airdrop-worthy action. Try uploading multiple file types for broader coverage.
